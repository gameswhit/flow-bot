# 🔄 FlowBot — Workflow Automation Engine

Visual workflow builder for automating business processes with AI decision nodes, human-in-the-loop, and 300+ integrations.

## Features

- 🔄 **Visual Builder** — Drag-and-drop workflow canvas with conditional logic, loops, error handling, and parallel branches.
- 🤖 **AI Decision Nodes** — Embed LLM-powered decisions in workflows — classify, extract, summarize, or route automatically.
- 👤 **Human-in-the-Loop** — Approval steps with Slack/Teams/email notifications, SLA tracking, and escalation rules.
- 📊 **Analytics** — Workflow execution analytics with bottleneck detection, SLA monitoring, and optimization suggestions.

## Quick Start

```bash
npm install -g flow-bot
flow-bot init my-project
cd my-project
flow-bot dev
```
